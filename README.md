# Pokémon: Type Quiz

A fast-paced Pokémon typing game where your knowledge is put to the test. Given a Pokémon type (or type combination),
your goal is to name as many matching Pokémon as possible before the timer runs out.

Built with **React**, **TypeScript**, **Vite**, and **Material UI**.

## Features

* ⚡ Fast, keyboard-first gameplay
* 🎯 Single and dual-type challenges
* 🔍 Smart autocomplete with fuzzy search
* 🔊 Optional Pokémon cry sound effects
* 📱 Responsive design
* ♿ Accessibility-focused interface

## How to Play

1. Start a new game.
2. A Pokémon type or type combination is shown.
3. Type the name of any Pokémon matching those types.
4. Correct answers increase your score and cannot be reused for the current challenge.
5. Beat the timer to reach the next challenge.
6. See how high you can score!

## Running Locally

### Requirements

* Node.js 22+
* pnpm 10+

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Preview

```bash
pnpm preview
```
